# QuakelikeBody
A vaguely quake-like character controller for Godot 3.

## License

Creative Commons Zero, 1.0. See repository. In other words: public domain.

## How to use

Drop `QuakelikeBody.gd` into your project, then use `QuakelikeBody` in the scene/node editor instead of `KinematicBody`. Make sure your scripts say `extends QuakelikeBody` instead of `extends KinematicBody`.
